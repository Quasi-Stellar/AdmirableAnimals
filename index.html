Admirable Animals

This is a project documenting ten rather different animal species:

Invertebrates:

  • The invasive Zebra Mussel (Dreissena polymorpha)
  • A really long tapeworm with no common name (Tetragonoporus calyptocephalus)
  • The minute Scarlet Pygmy Dragonfly (Nannophya pygmaea)
  • A tardigrade that survived exposure to outer space (Milnesium tardigradum)
  • The (somewhat unfairly) infamous Brown Recluse Spider (Loxosceles reclusa)

Vertebrates:

  • The invasive Northern Snakehead (Channa argus)
  • The smallest known vertebrate, which has no name (Paedophryne amauensis)
  • The Gyrfalcon, which flies horizontally at 90 mph (Falco rusticolus)
  • The (non-uniquely) frugivorous Pygmy Fruit Bat (Aethalops alecto)
